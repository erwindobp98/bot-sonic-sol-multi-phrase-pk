## Langkah-langkah

### 1. Clone Repositori

```bash
git clone https://github.com/Mnuralim/bot-sonic-sol-multi-phrase-PK.git
```

```
cd bot-sonic-sol
```

### 2. Instal Dependensi

```bash
npm install
```

### 3. Konfigurasi .env File

edit file .env dengan notepad atau run nano .env

```bash
nano.env
```

### 4. Run script

```bash
npm run start
```

\*pastikan udah install node js
